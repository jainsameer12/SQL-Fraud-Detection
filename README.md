# 💳 SQL Fraud Detection System

## 🚀 Overview
This project detects fraudulent transaction patterns using SQL-based feature engineering and machine learning.

## 🛠 Tech Stack
- SQL (MySQL)
- Python
- Pandas, Scikit-learn

## 🔍 Key Features
- Transaction pattern analysis
- Behavioral feature engineering
- Risk scoring system

## ⚙️ Workflow
1. Raw transaction data
2. SQL feature extraction
3. Data preprocessing
4. ML model training
5. Fraud prediction

## 📊 Example Insights
- Identified abnormal transaction spikes
- Detected device switching behavior
- Flagged high-risk users

## ▶️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/SQL-Fraud-Detection.git
cd SQL-Fraud-Detection
```

### 2. Create virtual environment (optional but recommended)
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the project

#### Option A: Run Jupyter Notebook
```bash
jupyter notebook
```
Open the notebook from the `notebooks/` folder.

#### Option B: Run Python scripts
```bash
python scripts/main.py
```

### 5. SQL Setup (if required)
- Import dataset into MySQL / PostgreSQL
- Execute SQL queries from the `sql/` folder

## 📌 Note
Dataset is not included due to size limitations. You can use your own dataset or generate synthetic data for testing.
